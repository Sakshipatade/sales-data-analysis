# Project Title: Sales Data Analysis

Description:
Analyzed retail sales dataset using Python to uncover key business insights related to revenue, profit, and customer segments.

Key Insights:

Identified top-performing sub-categories driving profit
Detected loss-making products affecting overall performance
Found negative impact of discounts on profitability
Analyzed regional and state-wise sales trends
Evaluated customer segment contribution

Tools Used:

-Python
-Pandas
-Matplotlib
