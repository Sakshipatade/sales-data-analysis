# Sales Data Analysis Project

## Overview

This project focuses on analyzing a retail sales dataset to uncover meaningful business insights related to revenue, profitability, customer segments, and product performance. The analysis helps in understanding key factors that impact business decisions.

---

## Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab

---

## Dataset Information

The dataset contains retail transaction data with the following key fields:

* Region, State, City
* Category, Sub-Category
* Sales, Profit, Discount
* Segment (Customer Type)

---

## Key Analysis Performed

### 1. Top Profitable Sub-Categories

Identified sub-categories contributing the highest profit to the business.

### 2. Loss-Making Sub-Categories

Analyzed products that generate losses despite having sales.

### 3. Discount vs Profit Analysis

Explored the relationship between discount levels and profitability.

### 4. State-wise Sales Analysis

Determined top-performing states based on total sales.

### 5. Segment-wise Sales Distribution

Evaluated revenue contribution from different customer segments.
